# Proyecto de Formulario usando Firebase

Este es un proyecto que utiliza **HTML**, **CSS**, **JavaScript**, **Bootstrap** y **Firebase** para generar la landing page de un formulario de Inicio de sesion, el formulario usa el correo Electronico del **usuario** para registrar a dicho usuario en la base de datos y de esa forma permitir su registro e ingreso
